# 🚀 Nexify – Premium Landing Page

A modern, fully responsive landing page for startups. Built with **Vanilla HTML, CSS & JavaScript**.

## ✨ Features

- 📱 **100% Responsive** – Optimized for all devices
- 🎯 **Sticky Navigation** with blur effect
- ✨ **Scroll Animations** 
- 📊 **Interactive Statistics** (Count-Up)
- ❓ **FAQ Accordion** 
- 🍔 **Mobile Menu** 
- 🎨 **Hover Animations**

## 🛠 Technologies

- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (Vanilla)
- Font Awesome Icons
- Google Fonts (Inter)
